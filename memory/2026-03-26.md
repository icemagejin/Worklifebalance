### 工作流程
1. **计算今日干支**
   - 年柱、月柱、日柱

2. **与大Boss八字合盘分析**
   - 大Boss八字：乙丑 丙戌 戊申
   - 分析吉凶运势

3. **生成对仗祝福语（6+6）**
   - 两句工整的祝福语，每句6个字
   - 根据今日干支寓意创作

4. **生成体现今日干支的背景图** ⭐
   - 工具：coze-image-gen
   - 命令：`node /workspace/projects/extensions/coze-openclaw-plugin/skills/coze-image-gen/scripts/gen.mjs --prompt "..." --size 2K`
   - 风格：现代国画写意，国画机制简约风，留白克制
   - 元素：一笔暖色光晕+几笔墨线
   - 约束：绝对禁止文字
   - prompt 示例：
     ```
     Modern Chinese ink painting style, minimalist composition with restrained negative space,
     one warm orange glow representing fire element, several light green ink strokes representing wood element,
     elegant and ethereal, no text, no characters, pure visual art, high quality
     ```

5. **合成完整海报**
   - 工具：Python PIL
   - 尺寸：1080x1920
   - 字体组合（重要）：
     - 超大数字：**Sans Bold**（粗体）
     - 祝福语/干支：**Serif Regular**（细体优雅）
     - 英文日期：Sans Regular（常规）
   - 超大数字（今日日期）
   - 极小字日期（MARCH SATURDAY）
   - 极小字干支
   - 两行6字对仗祝福语
