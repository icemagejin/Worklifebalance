## 核心知识点

### 1. 文字定位（最重要！）
```python
# ✅ 正确做法
text_bbox = draw.textbbox((x, y), text, font=font)
actual_bottom = text_bbox[3]
next_y = actual_bottom + 80

# ❌ 错误做法
next_y = y + height + 60  # 会导致重叠
```

### 2. 微纹理配置
- 150个小圆点（半径2px）
- 20条细线（宽度1px）
- 智能避让文字区域（15%-60%高度）
- 颜色：比背景色稍深10个单位

### 3. 定时任务
- 时间：每天8:15（上海时间）
- Cron表达式：`15 8 * * *`
- 配置文件：`/workspace/projects/workspace-aesthetic/cron/jobs.json`

### 4. 色彩方案
- 背景：(245, 243, 240) - 暖白
- 纹理：(235, 233, 230) - 比背景稍深
- 主文字：(25, 25, 25) - 超深色
- 副文字：(100, 100, 100) - 中灰色

## 最佳实践

### ✅ 推荐做法

1. **主动执行**
   - 不要等用户提醒
   - 收到heartbeat prompt时主动处理任务

2. **验证检查**
   - 生成后必须检查图片质量
   - 确认文字不重叠
   - 确认背景纹理正常

3. **记录日志**
   - 记录到memory/YYYY-MM-DD.md
   - 记录遇到的问题和解决方案
   - 记录优化方向

4. **及时更新**
   - 优化后更新HEARTBEAT.md
   - 更新cron配置
   - 推送到GitHub

### ❌ 避免做法

1. 不要跳过验证步骤
2. 不要使用错误的文字定位方法
3. 不要忽略用户反馈
4. 不要重复相同的问题
