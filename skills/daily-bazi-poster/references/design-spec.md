# 设计规范 - 微纹理海报

## 设计理念

现代、简洁、优雅，通过微妙的纹理增强视觉质感，同时保持信息的清晰易读性。

---

## 色彩系统

### 主色调
| 用途 | RGB | 说明 |
|------|-----|------|
| 背景色 | (245, 243, 240) | 暖白色，温和舒适 |
| 纹理色 | (235, 233, 230) | 比背景色稍深10个单位 |
| 主数字 | (25, 25, 25) | 超深色，强烈对比 |
| 副标题 | (100, 100, 100) | 中灰色，清晰但不抢眼 |

### 色彩原则
- **高对比度**：文字与背景对比度足够，确保可读性
- **微妙差异**：纹理与背景色差异极小，不抢焦点
- **温暖调性**：整体色调偏暖，营造舒适氛围
- **避免花哨**：不使用高饱和度颜色

### 字体选择
- **字体**：Noto Sans Regular
- **类型**：现代无衬线字体
- **格式**：TTC
- **路径**：`/usr/share/fonts/opentype/noto/NotoSansCJK-Regular.ttc`

### 字号配置
| 元素 | 计算方式 | 示例值 |
|------|---------|--------|
| 主数字 | height × 0.30 | 1920 × 0.30 = 576px |
| 副标题 | width × 0.04 | 1080 × 0.04 = 43px |

### 字体原则
- **现代感**：使用无衬线字体，简洁利落
- **层级清晰**：主数字远大于副标题
- **不使用粗黑体**：避免过于沉重
- **不使用衬线体**：保持现代感

## 布局规范

### 画布尺寸
- **宽度**：1080px
- **高度**：1920px
- **比例**：9:16（手机竖屏）

### 布局结构
```
┌─────────────────────┐
│                     │  (15% - 纹理区域)
│                     │
│                     │
│          4          │  (25% - 主数字起点)
│                     │
```

### 布局原则
- **居中对齐**：所有元素居中，视觉平衡
- **信息聚合**：日期信息聚合在一起，不分散
- **充足留白**：保留大量空白，呼吸感强
- **一眼看懂**：无需拼凑，直接理解

## 纹理规范

### 纹理配置
| 元素 | 数量 | 半径/宽度 | 颜色 | 分布 |
|------|------|----------|------|------|
| 小圆点 | 150 | 2px | (235, 233, 230) | 四周随机 |
| 细线 | 20 | 1px | (240, 238, 235) | 四周随机 |

### 纹理分布规则
```python
text_zone_top = height * 0.15      # 文字区域顶部边界
text_zone_bottom = height * 0.60   # 文字区域底部边界

# 纹理只分布在文字区域之外
if text_zone_top < y < text_zone_bottom:
    continue  # 跳过文字区域
```

### 纹理原则
- **不抢焦点**：纹理极淡，几乎看不见
- **有质感**：纸张的微妙纹理感
- **智能避让**：自动避开文字区域
- **随机分布**：每次生成略有不同

## 文字定位规范

### 正确的文字定位（重要！）

```python
# 1. 先计算绘制位置
text_bbox = draw.textbbox((x, y), text, font=font)

# 2. 获取实际底部位置
actual_bottom = text_bbox[3]

# 3. 下一行位置 = 实际底部 + 间距
next_y = actual_bottom + 80
```

### 常见错误（避免！）

```python
# ❌ 错误：使用字体高度计算
next_y = y + font_height + 60  # 会导致文字重叠

# ❌ 错误：直接使用估计值
next_y = y + 500  # 不精确，可能重叠或间距过大

# ✅ 正确：使用实际bottom位置
text_bbox = draw.textbbox((x, y), text, font=font)
next_y = text_bbox[3] + 80
```

### 间距规范
- **主数字到副标题**：80px
- **副标题行间距**：如果有多行，使用20px

## 设计禁忌

### ❌ 不要做的
- 不要使用粗黑体（太沉重）
- 不要使用衬线体（不够现代）
- 不要使用高饱和度颜色（太花哨）
- 不要让文字重叠（影响可读性）
- 不要让纹理太明显（抢焦点）
- 不要信息分散（难以理解）
- 不要缺乏留白（太拥挤）

### ✅ 要做的
- 使用现代无衬线字体
- 使用高对比度颜色
- 确保文字清晰无重叠
- 纹理微妙不抢眼
- 信息聚合居中
- 保留充足留白
- 一眼看懂

## 参考示例

### 标准海报
- 背景：暖白色 + 微纹理
- 主数字："4"（超大，居中）
- 副标题："4月4日 周六 · 2026"（居中，数字下方）
- 效果：现代、简洁、清晰

### 错误示例
- 背景：纯色，无纹理（太省事）
- 主数字："4"（太小，不够突出）
- 副标题："MAR TUE"（表达不自然）
- 文字重叠：副标题叠在数字下面（技术错误）
