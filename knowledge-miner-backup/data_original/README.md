## ID 格式

**格式**: `YYYYMMDD-角色名-序号`

保持与 Notion 思想母库一致，便于追溯。

- 日期: 年月日（如 20260331）
- 角色: 老周、小K、莫妮卡
- 序号: 两位数字（如 01）

## 每条记录格式
