# 🧪 Aesthetics Agent - 图片发送测试

## 测试目的
验证使用 `message` 工具发送图片到飞书群组的可行性。

### 2. 发送图片到美学群
使用 message 工具发送图片。

## 预期结果
✅ 图片成功发送到美学群

## 备选方案

如果 message 工具的 media 参数不可用，备选方案：

### 方案 A：上传到云空间，分享链接
1. 使用 feishu_drive_file 上传图片
2. 生成分享链接
3. 发送链接到群组

### 方案 B：插入到文档，分享文档
1. 创建飞书文档
2. 使用 feishu_doc.upload_image 插入图片
3. 分享文档到群组
