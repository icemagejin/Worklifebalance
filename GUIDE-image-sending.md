# 🎨 Aesthetics Agent - 飞书图片发送指南

## 概述

本指南说明如何将生成的图片发送到飞书群组。

## 工作流程

### Step 1: 生成图片（使用 coze-image-gen）

```bash
# 生成图片并保存到本地
coze-image-gen \
  --prompt "现代国画写意风格，暖橙色光晕，土黄色墨线，金色光泽，淡蓝水波纹，1080x1920" \
  --output /tmp/aesthetic-poster-$(date +%Y%m%d).png
```

**或者使用 exec 工具调用**：
```bash
exec: 
  command: coze-image-gen --prompt "..." --output /tmp/poster.png
```

### Step 2: 发送图片到飞书群组

使用 `message` 工具发送：

```json
{
  "action": "send",
  "channel": "feishu",
  "target": "chat:oc_95076f564f595dc80ae416c8221ad806",
  "message": "🎨 今日美学设计日报\n\n✅ 完成八字合盘海报设计\n✅ 创作现代国画写意风格背景图",
  "media": "/tmp/poster.png"
}
```

## 注意事项

### 图片格式要求
- **支持格式**: PNG, JPG, JPEG, GIF, WEBP
- **文件大小**: 最大 25MB
- **推荐尺寸**: 1080x1920（竖屏海报）

### 目标群组
- **美学群 ID**: `oc_95076f564f595dc80ae416c8221ad806`
- **格式**: `chat:{chat_id}`

### 文字内容
- 支持 Markdown 格式
- 建议包含设计说明和美学建议
- 可以使用表情符号增强视觉效果

## 故障排查

### 图片发送失败
1. 检查图片路径是否正确
2. 确认文件大小不超过 25MB
3. 检查图片格式是否支持

### 群组发送失败
1. 确认群组 ID 格式正确（`oc_xxx`）
2. 检查 Agent 是否有该群组的发送权限
3. 确认 `target` 参数格式为 `chat:{chat_id}`
