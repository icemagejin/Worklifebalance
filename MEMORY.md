## 2026-03-29 重大设计改进

### 问题
- 海报丑、排版差
- Python PIL 不懂设计
- 字体轻、只会居中、没有块面结构
- 反复调整3天，生成了9个版本

### 解决方案

#### 1. 动态布局
- 根据 VL 分析背景图形位置
- 选择布局类型（right_bottom/left_bottom/center/full）
- 文字避让图形

#### 2. 块面结构
- 不只是居中，要有空间划分
- 上半块面：超大数字
- 中间块面：农历+干支（偏左或偏右）
- 下半块面：利弊

#### 3. 字体力度
- 中文用粗黑体（Noto Sans Bold/Black）
- 不要细宋体（太轻易）
- 字体要有力量感

#### 4. 动态字号
- 根据背景图形空间调整字号
- 根据文字内容长度调整字号
- 不是固定搭配，是动态组合

### 参考图分析（用户提供的4张优秀设计）

**共同特点**：
1. 超大数字（占画面30-40%）
2. 留白充足（>60%）
3. 字体对比强（粗黑数字 + 有力中文）
4. 块面结构清晰
5. 背景轻盈不抢焦点
6. 装饰极简
