## 2. 获取宜忌数据

### 方法：搜狐运势抓取

```bash
curl -s "https://m.sohu.com/a/[文章ID]/" | grep -E "宜|忌"
```

### 今日（2026-03-29）宜忌

**八字**：丙午年·辛卯月·壬寅日

**忌**：嫁娶、祈福、出火、入宅

## 3. 改写成6+6对仗福语

### 原则
- 从"宜"中选2-4个关键词，组合成6字
- 从"忌"中选2个关键词，组合成6字
- 两句对仗，用·连接

### 今日示例

**宜** → "立券交易 · 修造动土"

**忌** → "嫁娶择日 · 祈福缓行"

### 其他示例

- 宜：开业、出行、嫁娶 → "开业出行 · 宜嫁娶"
- 忌：安床、移徙 → "安床暂缓 · 移徙待时"

## 4. 生成海报

### 版式结构

```

│  2026.03  壬寅  29  │  ← 左上年月，右侧日柱，中间日期数字（最大）
│                     │
│    [大面积留白]     │
│                     │
│  立券交易 · 修造动土 │  ← 宜
│  嫁娶择日 · 祈福缓行 │  ← 忌

```

## 技术要点

1. **八字计算**：年月日干支需准确（之前算错了月份）
2. **宜忌来源**：从搜狐/网易等运势文章中提取真实数据
3. **改写技巧**：
   - 宜：选积极动词，用·连接
   - 忌：用"择日"、"缓行"、"暂缓"等委婉表达
4. **对仗要求**：前后各6字，语法对称

## 每日执行流程

1. 计算今日八字（年月日三柱）

4. 改写成6+6对仗福语
5. 生成诧寂风背景图
6. 合成1080x1440海报
7. 发送到美学群
8. 更新日记记录
