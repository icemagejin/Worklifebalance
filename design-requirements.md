# 八字美学海报设计要求

## 版式结构

### 整体规格
- **尺寸**: 1080x1920 (9:16 竖版)
- **风格**: 现代国画写意 + 小清新美学
- **配色**: 暖橙色 + 土黄色 + 金色 + 淡蓝

## 布局分层（从上到下）

### 1️⃣ 顶部区域（10%）
```
┌─────────────────────┐
│  每日美学设计日报     │  ← 小标题，浅色
│  2026年3月29日       │  ← 日期，细字体
│                     │
│  丙午·己丑·庚子       │  ← 核心干支，醒目
└─────────────────────┘
```

**要求**:
- "每日美学设计日报": 小号字，浅色调
- 日期: Serif Regular，优雅
- 干支标题: Sans Bold，最大字号，视觉焦点

### 2️⃣ 中间视觉区域（60%）
```
┌─────────────────────┐
│                     │
│   [背景图留白区域]   │  ← 写意背景
│                     │  ← 半透明白色遮罩
│   [视觉呼吸空间]     │
│                     │
└─────────────────────┘
```

**要求**:
- 现代国画写意风格背景
- 柔和水墨晕染，光晕效果
- 大量留白，增加呼吸感
- 半透明白色遮罩（透明度60-70%）

### 3️⃣ 底部信息区域（30%）
```
┌─────────────────────┐
│  ✅ 完成任务列表     │  ← 图标+文字
│                     │
│  设计元素           │
│  🌅 配色：[四元素]   │
│  🖊️ 线条：简约墨线   │
│  ✨ 氛围：[描述]     │
│                     │
│  💡 美学建议         │
│  • 建议1             │
│  • 建议2             │
│                     │
│  🖼️ 设计作品         │
│  • 名称：[作品名]    │
│  • 风格：现代国画    │
│  • 思路：[理念]     │
└─────────────────────┘
```

**要求**:
- 字体大小适中，易读
- 使用emoji作为视觉引导
- 分区块清晰：任务 → 元素 → 建议 → 作品
- 文字密集但有序，不拥挤

## 视觉设计规范

### 背景图要求
- **风格**: 现代国画写意
- **主题**: 对应今日四元素（火土金水）
- **色彩**: 暖橙色光晕 + 土黄色 + 金色光泽 + 淡蓝水波纹
- **质感**: 柔和晕染，光晕效果
- **留白**: 40%以上留白空间

### 文字规范
- **干支标题**: Sans Bold, 最大字号
- **日期/小标题**: Serif Regular, 优雅
- **正文**: Sans Regular, 清晰易读
- **颜色**: 白色为主，淡蓝色点缀

### 装饰元素
- 简约国画墨线（土黄色）
- 点状光晕（金色）
- 柔和水波纹（淡蓝）
- 避免过于复杂的图形

## 4步工作流

### Step 1: 获取今日干支
- 计算八字（年月日）
- 识别火土金水四元素

### Step 2: 生成背景图
- 使用 coze-image-gen
- 提示词模板: "现代国画写意风格，[今日元素]主题，暖橙土黄金色淡蓝配色，1080x1920"
- 智能裁剪适配海报比例

### Step 3: 合成海报
- 添加白色半透明遮罩（60-70%透明度）
- 按照版式层次添加文字
- 字体粗细对比（Bold + Regular）
- 大量留白，增加呼吸感

### Step 4: 发送到美学群
- 包含海报图片
- 附上设计说明
- 给出美学建议

## 质量检查清单

每次完成时检查:
- [ ] 尺寸是否为 1080x1920
- [ ] 三个区域层次分明
- [ ] 干支标题醒目突出
- [ ] 背景图留白足够（>40%）
- [ ] 文字清晰可读
- [ ] 配色和谐（暖橙土黄金色淡蓝）
- [ ] 国画写意风格明显
- [ ] 没有过度装饰
- [ ] 整体呼吸感良好

## 设计理念核心

> **少即是多，留白即是美**

- 简约不等于简单
- 留白让作品有呼吸感
- 现代与传统的融合
- 小清新的治愈感
